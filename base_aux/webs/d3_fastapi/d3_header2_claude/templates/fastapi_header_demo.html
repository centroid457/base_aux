FastAPI Header Widget

v1.0 zero deps

One <script> tag. Auto-reads /openapi.json and renders a sticky service header — no build step, no dependencies, works in any FastAPI project.

// play with the widget above

env
status
service

// minimal usage

<!-- In your Jinja template or any HTML -->
<script src="/static/fastapi-header.js"></script>

// with options

<script
  src="/static/fastapi-header.js"
  data-env="production"
  data-theme="dark"
  data-position="top"
  data-collapsed="false"
></script>

// fastapi static mount

# main.py
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles

app = FastAPI()
app.mount("/static", StaticFiles(directory="static"), name="static")
# Put fastapi-header.js inside ./static/

// features

⚡
Auto-reads /openapi.json
Picks up service name, version, description, and all routes automatically.
🩺
Health check
Pings /health on load. Falls back gracefully if the endpoint is missing.
🌍
Env detection
Auto-detects dev / staging / prod from hostname, or override via data-env.
📋
Route explorer
Click the routes pill to expand all endpoints with color-coded method badges.
🔗
Docs links
One-click to /docs (Swagger) and /redoc — always in reach.
⌨️
Alt+H shortcut
Keyboard toggle to collapse/expand the header without touching the mouse.
