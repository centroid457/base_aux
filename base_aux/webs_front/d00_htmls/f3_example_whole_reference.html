типовые

strong — важный текст (жирный + смысловая важность).

em — акцентирование (курсив + логическое ударение).

b — просто жирный, без дополнительной важности (визуально).

i — просто курсив (технический термин, иноязычие).

mark — выделенный цветом (как маркер).

small — мелкий примечательный текст (лицензия, сноска).

удалённый текст и вставленный (обычно используется в редактурах).

H2O — подстрочный (sub), E = mc2 — надстрочный (sup).

code example — для фрагментов кода внутри текста и в сочетании с PRE.

С уважением,

<p> — для смыслового разделения мыслей (блочный элемент с отступами).
<br> — для переноса внутри одного абзаца (строчный разрыв). Не используйте <br> для создания пустых мест или разделения разных идей.

📑 Вкладки (табы) — живой компонент

Переключение панелей: только одна активная вкладка

Вкладки (tabs) — это элемент интерфейса, который позволяет переключаться между несколькими панелями с контентом, не перезагружая страницу.

В чистом HTML нет специального тега для табов, но их легко реализовать с помощью CSS и JavaScript (или только CSS с радиокнопками).

✔️ Экономит место на странице
✔️ Улучшает навигацию по группам информации

<div class="tabs-buttons">
  <button class="tab-btn active" data-tab="tab1">Вкладка 1</button>
  <button class="tab-btn" data-tab="tab2">Вкладка 2</button>
</div>
<div class="tabs-content">
  <div class="tab-pane active" id="tab1">...</div>
  <div class="tab-pane" id="tab2">...</div>
</div>
// JS: при клике добавляем/удаляем класс .active

В этом примере используется простой JavaScript: при клике на кнопку активируется соответствующий .tab-pane и меняется стиль кнопки.

  • ⚠️ Важно добавлять атрибуты role="tablist", aria-selected для скринридеров (в учебном примере опущено для краткости, но в реальном проекте используйте).
  • 🧩 Можно сделать и на чистом CSS: <input type="radio"> + :checked, но JS даёт больше контроля и динамики.
  • 🎨 Активная вкладка должна быть визуально выделена (цвет, подчёркивание).
  • 📱 На мобильных устройствах табы удобно превращать в аккордеон (но это уже продвинутый вариант).
🧩 Суть: табы — это не встроенный тег, а популярный паттерн. Здесь показана классическая реализация на JS + CSS.
Альтернатива — CSS-табы на радиокнопках (без JS)
✨ Это вкладки без единой строчки JavaScript! Используются <input type="radio"> и CSS-селектор :checked ~ ....
🧠 Приём: скрытые радиокнопки, лейблы как кнопки, а содержимое переключается через общий контейнер и display: none/block.
✔️ Минус: сложнее управлять динамическим добавлением вкладок. Плюс: работает без JS, очень легковесно.
📌 Чисто CSS-вкладки — хороший вариант для статичных страниц, где не нужно добавлять вкладки динамически.

🧠 Как это работает (Flexbox): 
    <div style="display: flex; justify-content: space-between; align-items: center;">
        <div>Левый элемент</div>
        <div style="display: flex; gap: 1rem;">Центральные элементы</div>

    </div>
✔️ justify-content: space-between — распределяет всё пространство, прижимая левый и правый блоки к краям, а центр — ровно посередине.
✔️ Внутри каждой группы — свой flex или просто блочные элементы.
✔️ gap задаёт отступы между элементами внутри группы.
✔️ align-items: center — вертикальное выравнивание по центру.
💡 justify-content: space-between, space-evenly, center — основные приёмы.
